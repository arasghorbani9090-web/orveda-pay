# Orveda Pay

Premium, enterprise-grade fintech marketing website for **Orveda Pay** — global financial infrastructure for modern businesses.

Built with **Next.js 14 (App Router)**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Features

- Ultra-modern, trust-focused fintech design with a professional navy / royal-blue / emerald palette and gold highlights
- Fully responsive (desktop, tablet, mobile) with elegant scroll + page-transition animations
- SEO-optimized (per-page metadata, Open Graph, sitemap, robots) and accessible (skip link, focus states, semantic markup)
- Static-rendered pages for fast loading

## Pages

| Route | Description |
| --- | --- |
| `/` | Home — hero, dashboard mockup, trust, services, how-it-works, security |
| `/solutions` | Solutions overview + industries |
| `/multi-currency-accounts` | 8-currency accounts (USD, EUR, GBP, AED, CAD, AUD, CHF, SGD) |
| `/international-payments` | SWIFT, cross-border settlement, collections |
| `/compliance-security` | KYC, KYB, AML, encryption, risk & fraud |
| `/pricing` | Plans + FAQ |
| `/about` | Mission, vision, values, careers |
| `/contact` | Contact form, support channels, map |
| `/register` | Multi-step business onboarding portal with drag & drop document upload |
| `/dashboard` | Client dashboard UI mockup |
| `/legal/privacy`, `/legal/terms`, `/legal/aml` | Legal documents |

## Getting started

```bash
npm install
npm run dev      # http://localhost:3000
```

## Build

```bash
npm run build
npm run start
```

## Project structure

```
app/                 # App Router pages, layout, globals.css, sitemap, robots
components/          # Header, Footer, Logo, reusable sections, icons, motion
tailwind.config.ts   # Brand theme (colors, fonts, shadows, animations)
```
